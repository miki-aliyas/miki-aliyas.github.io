おみくじアプリ

あなたの今日の運勢をワンクリックで占える。
LINEのAPIを使用しておみくじ結果がスマートフォンに届く機能を実装。

制作期間：

2ヶ月

使用言語、ツールなど：

Android studio / Java

バージョン管理：

GitHub

ソースコード：

https://github.com/miki-aliyas/Today-sFortuneApp

操作方法

1.「おみくじを引く」をタップ

2.結果が表示される。「トップページに戻る」をタップ

3.自身のスマートフォンのLINEにおみくじ結果が届く
